This guide will assist employees on finding appropriate training curriculum for the Azure cloud offering.

 

Microsoft Azure (Beginners):  Microsoft Azure is a comprehensive public cloud offering. It includes virtual machine hosting for Windows Servers, unstructured storage, table storage and hosted SQL databases. This series of courses (10 hours) covers resource creation and administration on Microsoft’s public cloud.

 

Click here to find the Microsoft Azure courses in DXC University.

·        For the entry level Application Developer, complete this entry-level course which provides an introductory guide to Azure, Node.JS, and SQL Server. (sl_adnj_a03_it_enus)

 

Developing Microsoft Azure and Web Services (Certification Prep): This curriculum (19.5 hours) will provide developers with more advanced fundamentals of Azure and Web Services as well as those preparing for Microsoft Exam 70-487. In this folder, you will see 8 courses, a Test Prep and CSC University Mentoring Service available for Microsoft Azure assistance.

 

Click here to find the Developing Microsoft Azure and Web Services courses in DXC University.

 

 

Developing Microsoft Azure Solutions (Experienced): Microsoft Azure is a subscription-based collection of interrelated cloud-hosted services that supports the development of secure, salable, and flexible cloud-hosted application and storage solutions. Understanding the depth and breadth of the Azure collection is key to developing durable, secure, resilient cloud-hosted, in-house, client, and public solutions. The 70-532 examination is designed to test a candidate's proficiency in developing such solutions on the Azure platform. The individual courses within this learning path (7 hours) will assist candidates who are studying for and preparing to take the Microsoft Exam, 70-532: Developing Microsoft Azure Solutions.

Click here to find the Developing Microsoft Azure Solutions courses in DXC University.

 

 

Microsoft Azure: Implementing Infrastructure Solutions (Experienced): Azure is Microsoft's cloud, which integrates compute, storage, data, networking and app in an open, flexible, and scalable platform.This learning path (15 hours) of Skillsoft courses covers the implementation of these services as holistic solutions. These courses cover the objectives for Microsoft Exam 70-533: Implementing Microsoft Azure Infrastructure Solutions. This exam completes the requirement for Microsoft Specialist certification in Microsoft Azure.

Click here to find the Microsoft Azure: Implementing Infrastructure Solutions courses in DXC University.

 

 

Custom Azure Courses: These self-study courses with no audio (estimated 36 hours) will provide application developers or technical architects with an introduction of Microsoft Azure and guidelines of how to develop a cloud application. These lessons include content, demos, previews and reference materials.

Click here to find the Custom Azure courses in DXC University.
